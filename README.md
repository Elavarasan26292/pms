## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov



Please make sure you have the mongodb working in your locan environment

Application Flow

1.Create user
2.Login and get the access token
3.Pass access token to subsequest api requests
```
